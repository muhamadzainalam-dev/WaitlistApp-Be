Backend API for waitlist signups, built to handle integrations like email delivery and messaging for onboarding flows.

> 🔗 **Frontend Repo:** `muhamadzainalam-dev/WaitlistApp-Fe`  
> `https://github.com/muhamadzainalam-dev/WaitlistApp-Fe`

---

## 💼 Skills & Tech Used

- **Server:** Node.js, Express
- **Email Providers / APIs:** Sendinblue (`@sendinblue/client`), Resend (`resend`), Nodemailer
- **Messaging:** Twilio
- **Config & Middleware:** dotenv, cors
- **Dev Script:** `nodemon` (used in `npm run dev`)

---

## 🚀 Run Locally

```bash
git clone https://github.com/muhamadzainalam-dev/WaitlistApp-Be.git
cd WaitlistApp-Be
npm install
npm run dev
```
